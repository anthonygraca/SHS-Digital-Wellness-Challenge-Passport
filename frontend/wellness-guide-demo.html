US-16 Implementation

🩺 Conversational Wellness Guide

A themed in-app assistant that answers wellness questions grounded in SHS content, nudges next tasks, and links campus resources. With safety guardrails (US-17) for crisis routing and medical advice refusal.

✨ Features Implemented

  • Themed wellness guide with persona support
  • Grounded answers from SHS-approved content
  • Crisis detection and routing (988, campus counseling, SHS)
  • Medical advice refusal (diagnosis, dosage, treatment)
  • Out-of-scope deflection with resource links
  • Minimal conversation logging (FR-E6: 24hr retention, no PHI)
  • Chat UI with FAB launcher and responsive design

📡 API Endpoints Created

POST /api/guide/messages - Send message to guide (with guardrails)

POST /api/conversations/sessions - Create/get conversation session

GET /api/conversations/sessions/current - Get active session

GET /api/guide/crisis-resources - Get crisis resources
